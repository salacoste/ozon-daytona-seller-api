# Ozon Seller API SDK - Бизнес-обзор модулей

## Краткое описание

TypeScript SDK для работы с API Ozon для продавцов. Предоставляет доступ к 263 методам API через 32 модуля для управления всеми аспектами бизнеса на маркетплейсе Ozon.

---

## 🚀 **ОСНОВНЫЕ МОДУЛИ (P0 Priority)**

### 1. **FBS API** - Управление продажами со склада продавца
**Что это:** Полное управление заказами, которые отправляются с вашего склада

**Основные функции:**
- 📦 **Управление заказами** - получение списка заказов, просмотр деталей, изменение статусов
- 🚚 **Отгрузка товаров** - создание актов приема-передачи, печать документов
- 📋 **Работа с накладными** - получение и обновление данных об отгрузках
- 🌍 **Международная доставка** - управление ограничениями по странам
- 📊 **Аналитика продаж** - статистика по заказам и отгрузкам

**Бизнес-ценность:** Автоматизация всего процесса обработки заказов FBS

---

### 2. **FBO API** - Управление товарами на складах Ozon
**Что это:** Работа с товарами, которые хранятся на складах Ozon

**Основные функции:**
- 📦 **Поставки товаров** - создание и отслеживание поставок на склады Ozon
- 📋 **Управление заказами** - просмотр и обработка заказов FBO
- 🏷️ **Этикетки и штрихкоды** - генерация этикеток для товаров
- 📊 **Отчёты по складам** - аналитика по остаткам и движению товаров
- 💰 **Финансовая отчётность** - доходы и расходы по FBO

**Бизнес-ценность:** Эффективное управление товарами на складах Ozon

---

### 3. **ProductAPI** - Управление каталогом товаров
**Что это:** Создание, редактирование и управление карточками товаров

**Основные функции:**
- 🛍️ **Создание товаров** - добавление новых товаров в каталог
- ✏️ **Редактирование карточек** - обновление описаний, фото, характеристик
- 👀 **Просмотр товаров** - получение информации о товарах
- 🔄 **Массовые операции** - импорт и обновление больших объёмов товаров
- 📈 **Аналитика товаров** - статистика просмотров и продаж

**Бизнес-ценность:** Полный контроль над каталогом товаров

---

### 4. **Prices&StocksAPI** - Управление ценами и остатками
**Что это:** Контроль цен и количества товаров на складах

**Основные функции:**
- 💰 **Управление ценами** - установка и обновление цен товаров
- 📦 **Контроль остатков** - отслеживание количества товаров на складах
- 📊 **Аналитика остатков** - детальная статистика по складам
- ⚡ **Массовые обновления** - быстрое изменение цен и остатков
- 🔔 **Уведомления** - оповещения о низких остатках

**Бизнес-ценность:** Оперативное управление ценообразованием и логистикой
