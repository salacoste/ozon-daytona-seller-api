# CategoryAPI - Работа с категориями товаров

## 🎯 Что это такое

CategoryAPI предоставляет инструменты для работы с товарными категориями платформы Ozon. Помогает правильно категоризировать товары, изучать требования категорий и оптимизировать размещение продукции для максимальной видимости.

## 💼 Основные возможности

### 🏷️ **Справочник категорий**
- Полное дерево категорий Ozon с иерархией
- Описания и характеристики каждой категории
- Требования к товарам в категориях
- Ограничения и специальные условия

### 🔍 **Поиск и навигация**
- Поиск категорий по ключевым словам
- Навигация по дереву категорий
- Получение родительских и дочерних категорий
- Рекомендации категорий для товаров

### 📊 **Атрибуты категорий**
- Обязательные характеристики товаров
- Дополнительные атрибуты и фильтры
- Допустимые значения характеристик
- Форматы и ограничения атрибутов

### 🎯 **Оптимизация размещения**
- Анализ популярности категорий
- Конкурентность и насыщенность категорий
- Рекомендации по выбору оптимальной категории
- Тренды развития категорий

## 🚀 Типовые сценарии использования

### Правильная категоризация нового товара
```
1. Анализ характеристик товара
2. Поиск подходящих категорий
3. Сравнение требований категорий
4. Выбор оптимальной категории для размещения
```

### Аудит существующих размещений
```
1. Проверка соответствия товаров категориям
2. Выявление неоптимальных размещений
3. Анализ конкурентности текущих категорий
4. Перемещение товаров в более подходящие категории
```

### Планирование ассортимента
```
1. Изучение популярных категорий
2. Анализ требований к новым товарам
3. Поиск незаполненных ниш
4. Планирование развития товарных линеек
```

## 💡 Рекомендации по использованию

### Выбор категории
- **Соответствие продукта** - товар должен точно соответствовать описанию категории
- **Конкурентность** - учитывайте насыщенность категории конкурентами
- **Популярность** - размещайте в категориях с высоким трафиком покупателей
- **Требования** - убедитесь, что можете выполнить все требования категории

### Оптимизация видимости
- Заполняйте все обязательные атрибуты товаров
- Используйте популярные ключевые слова в названиях
- Регулярно проверяйте изменения требований категорий
- Адаптируйте описания под специфику категории

### Работа с атрибутами
- Изучите все доступные характеристики категории
- Заполняйте дополнительные атрибуты для лучшей фильтрации
- Используйте стандартные значения из справочников
- Регулярно обновляйте характеристики товаров

**Бизнес-ценность:** Правильная категоризация товаров для максимальной видимости и соответствия требованиям
