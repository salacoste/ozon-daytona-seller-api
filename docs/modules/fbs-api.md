# FBS API - Продажи со склада продавца

## 🎯 Что это такое

FBS (Fulfillment by Seller) - это модель продаж, при которой товары хранятся на вашем складе, а вы самостоятельно обрабатываете и отправляете заказы покупателям.

## 💼 Бизнес-задачи которые решает

- **Полный контроль логистики** - управление всем процессом от заказа до доставки
- **Снижение затрат** - экономия на хранении товаров на складах Ozon
- **Быстрая отгрузка** - возможность отправки товаров в день заказа
- **Гибкость ассортимента** - нет ограничений по количеству и типам товаров

---

## 📋 Основные функции

### 1. **Управление заказами**
**Что делает:** Получение и обработка всех заказов FBS

**Методы:**
- `listOrders()` - получить список всех заказов
- `getOrderDetails()` - детальная информация по заказу
- `updateOrderStatus()` - изменить статус заказа

**Бизнес-ценность:** 
- Автоматический мониторинг новых заказов
- Отслеживание статусов обработки
- Планирование отгрузок

---

### 2. **Создание отгрузок**
**Что делает:** Формирование актов приема-передачи для отправки товаров

**Методы:**
- `createAct()` - создать акт приема-передачи
- `getActInfo()` - получить информацию об акте
- `checkUnrealizedOrders()` - проверить нереализованные заказы

**Бизнес-ценность:**
- Автоматизация оформления документов
- Контроль готовности к отгрузке
- Соответствие требованиям Ozon

---

### 3. **Управление документами**
**Что делает:** Работа с накладными и сопроводительными документами

**Методы:**
- `getManifest()` - получить манифест отгрузки
- `getShipmentLabels()` - этикетки для посылок
- `getPackageLabels()` - этикетки для упаковки

**Бизнес-ценность:**
- Правильное оформление отгрузок
- Соблюдение стандартов маркировки
- Избежание штрафов за неправильные документы

---

### 4. **Географические ограничения**
**Что делает:** Управление доставкой по странам и регионам

**Методы:**
- `getCountryList()` - список доступных стран
- `setCountryRestrictions()` - установить ограничения доставки
- `getDeliveryRestrictions()` - текущие ограничения

**Бизнес-ценность:**
- Контроль географии продаж
- Соблюдение международных требований
- Оптимизация логистических затрат

---

### 5. **Аналитика и отчёты**
**Что делает:** Статистика по заказам и эффективности FBS

**Методы:**
- `getOrdersAnalytics()` - аналитика заказов по периодам
- `getShipmentStatistics()` - статистика отгрузок
- `getPerformanceMetrics()` - показатели эффективности

**Бизнес-ценность:**
- Оценка эффективности FBS
- Выявление проблемных зон
- Планирование развития

---

## 📊 Ключевые метрики

- **Время обработки заказа** - от получения до отгрузки
- **Процент брака** - доля возвращённых товаров
- **География продаж** - охват регионов доставки
- **Рентабельность FBS** - прибыль с учётом логистических затрат

---

## 💡 Типовые сценарии использования

### Сценарий 1: Ежедневная обработка заказов
1. Получить список новых заказов за день
2. Проверить наличие товаров на складе
3. Сформировать акт приема-передачи
4. Распечатать этикетки и документы
5. Обновить статусы заказов при отгрузке

### Сценарий 2: Настройка географии доставки
1. Анализировать спрос по регионам
2. Оценить рентабельность доставки
3. Настроить ограничения по странам
4. Мониторить изменения в продажах

### Сценарий 3: Контроль качества логистики
1. Отслеживать время обработки заказов
2. Анализировать причины возвратов
3. Выявлять проблемы в цепи поставок
4. Оптимизировать процессы отгрузки

---

## ⚠️ Важные особенности

- **Сроки обработки** - Ozon требует обработки заказов в течение 24 часов
- **Штрафы** - за просрочку обработки могут начисляться штрафы
- **Документооборот** - все отгрузки должны сопровождаться правильными документами
- **Международная доставка** - требует соблюдения таможенного законодательства
