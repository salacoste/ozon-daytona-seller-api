# PromosAPI - Управление акциями и скидками

## 🎯 Что это такое

PromosAPI предоставляет полный набор инструментов для создания, управления и анализа промо-акций на платформе Ozon. Помогает увеличить продажи через эффективные маркетинговые кампании.

## 💼 Основные возможности

### 🎉 **Создание акций**
- Скидки на отдельные товары
- Бандлы и комплекты товаров  
- Промокоды для покупателей
- Программы лояльности

### 💰 **Управление скидками**
- Процентные и фиксированные скидки
- Прогрессивная система скидок
- Скидки для VIP-клиентов
- Временные ограничения по акциям

### 📅 **Планирование промо**
- Календарь акций и распродаж
- Сезонные кампании
- Автоматический запуск акций
- Координация с маркетинговыми событиями Ozon

### 📊 **Аналитика эффективности**
- ROI промо-кампаний
- Конверсия по акциям
- Дополнительные продажи
- Влияние на лояльность клиентов

### 🎯 **Таргетинг**
- Персональные предложения
- Сегментация по поведению клиентов
- Ретаргетинг по корзине
- Геотаргетинг по регионам

**Бизнес-ценность:** Увеличение продаж на 15-40% через эффективные промо-кампании
